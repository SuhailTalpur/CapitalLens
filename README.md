CapitalLens is a full-stack MERN (MongoDB, Express, React, Node.js) fintech application that combines real-time stock data, AI-powered insights, and financial news into a single powerful dashboard.

It integrates stock market data from Finnhub API and news from NewsAPI to help users track markets, analyze trends, and stay updated with global financial events.

---

## ✨ Features

### 📊 AI Stock Market Insights
- Real-time stock data using Finnhub API
- Price trends and analytics
- AI-based insights (trend summaries, predictions)
- Company-specific stock analysis

### 📰 Financial News Aggregator
- Latest finance and business news via NewsAPI
- Category filtering (crypto, stocks, startups, economy)
- Search functionality
- Real-time updates

### 📈 Dashboard & Visualization
- Interactive charts (Chart.js / Recharts)
- Market overview
- Trending stocks
- Clean fintech UI

### 🔐 User Features (Optional)
- Authentication (JWT / Clerk)
- Watchlist / favorite stocks
- Personalized dashboard
